# Inclusion and Exclusion Criteria

## Inclusion Criteria (IC)

| ID  | Criteria                                                                              |
| --- | ------------------------------------------------------------------------------------- |
| IC1 | Paper is written in English                                                           |
| IC2 | Published in 2018 or later                                                            |
| IC3 | Peer-reviewed journal, conference, or workshop paper                                  |
| IC4 | Related to LLMs, automated test generation, unit test generation, or software testing |
| IC5 | Contains empirical evaluation or experimental results                                 |

A paper is included only if all IC criteria are satisfied.

## Exclusion Criteria (EC)

| ID  | Criteria                                                                          |
| --- | --------------------------------------------------------------------------------- |
| EC1 | Duplicate paper                                                                   |
| EC2 | Full-text PDF is unavailable                                                      |
| EC3 | Poster, tutorial, keynote, extended abstract, or paper shorter than 4 pages       |
| EC4 | Focuses only on test execution, maintenance, or debugging without test generation |
| EC5 | Not related to software testing or test generation                                |
| EC6 | Survey, opinion, editorial, or position paper without experiments                 |
| EC7 | Non-English paper                                                                 |

A paper is excluded if any EC criterion is satisfied.
